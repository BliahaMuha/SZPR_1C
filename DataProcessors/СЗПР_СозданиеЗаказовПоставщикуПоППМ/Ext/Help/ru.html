Обработка Создание заказов поставщику по ППМ

Обработка предназначена для создания заказов поставщику по данным документа ППМ.

Обработка содержит табличную часть со следующими полями:

  • Дата поступления – дата начала периода, по которым рассчитана потребность в материалах. Заполняется автоматически по данным документа ППМ, недоступна для изменения;
  • Номенклатура – наименование материала, заполняется автоматически по данным документа ППМ, недоступна для изменения;
  • Характеристики – наименование характеристики номенклатуры, заполняется автоматически по данным документа ППМ, недоступна для изменения;
  • Ед.изм. – единица измерения номенклатуры, заполняется автоматически по данным документа ППМ, недоступна для изменения;
  • Количество – количество материала, заполняется автоматически по данным документа ППМ, недоступно для изменения;
  • Контрагент – поставщик материалов, выбирается из справочника Контрагенты;
  • Вид цены – выбирается из перечня Виды цен номенклатуры поставщиков;
  • Цена – заполняется автоматически из выбранного вида цен или проставляется вручную;
  • Сумма – рассчитывается автоматически как произведение количества на цену.

В табличной части необходимо отметить позиции материалов, которые планируется заказать

Формирование заказов поставщикам производится в разбивке по контрагентам.

Если в верхней части обработки отмечено галочкой Формировать заказы по датам поступления, то заказы поставщикам сформируются еще и в разбивке по датам.

По кнопке Сформировать заказы создаются документы Заказ поставщику.

Важно! Обработка Создание заказов поставщику по ППМ создает заказы поставщику на то количество материала, которое указано в документе ППМ, без учета остатков на складе и ранее заказанных материалов. Обработка не контролирует количество созданных на основании выбранного ППМ заказов поставщику.

На закладке Сформированные документы можно посмотреть список сформированных заказов поставщику.

Заказы создаются в состоянии не проведены. Необходимо после создания заказов убедиться, что документ заполнился корректно, и провести документ.
